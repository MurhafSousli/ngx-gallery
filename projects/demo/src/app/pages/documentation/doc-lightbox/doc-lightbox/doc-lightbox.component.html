Usage
1

Import BrowserAnimationsModule and HttpClientModule in the root module

2

Import GalleryModule and LightboxModule

Use {{ 'LightboxModule.withConfig({ ... })' }} to set config that applies on the module and its children

3

See Lightbox Example

To set global config that applies on all the lightbox overlays across the app, provide the config value using LIGHTBOX_CONFIG token.
