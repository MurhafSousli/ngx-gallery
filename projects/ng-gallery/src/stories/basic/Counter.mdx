# Gallery Counter

Indicates the number of the active value and the total amount of items in the gallery.

<Canvas of={GalleryStories.CounterExample}/>



## Counter position

There are 2 choices for positioning the counter, `top` and `bottom`.

**Example**

```html
<gallery counter counterPosition="bottom" />
```



## List of options related to gallery counter

- counter
- counterPosition
