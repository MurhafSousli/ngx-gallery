Custom Templates Example

Overview

This package provides a custom set of directives to make it easy to customize different gallery templates.

These directives are: *galleryItemDef, *galleryImageDef, *galleryThumbDef and *galleryBoxDef

Passing in data to a custom template

When using one of the previous templates above, additional context data can be used to render the data as needed.

Available data:

  • let item Object of the item, including properties like src, thumband alt ... etc.
  • let index = index Index of the item.
  • let type = type Type of the item, default ones are image, video, youtube and iframe.
  • let active = active True if this item is the active one.
  • let count = count The number of total items.
  • let first = first True if this item is first.
  • let last = last True if this item is last.
